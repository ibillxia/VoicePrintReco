A discription of the file contents:

ex1_01.wav	- a 1 Second speech file, in PCM 01 byte format.  The
		  same speech as ex1_10.wav
ex1_01sh.wav	- a shortpack'd version of ex1_01.wav
ex1_01st.wav	- a shorten'd version of ex1_01.wav
ex1_01wp.wav	- a awavpack'd version of ex1_01.wav

ex1_10.wav	- a 1 Second speech file, in PCM 10 byte format.  The
		  same speech as ex1_01.wav
ex1_10.wvc      - a corrupted version of ex1_10.wav
ex1_10sh.wav	- a shortpack'd version of ex1_10.wav
ex1_10st.wav	- a shorten'd version of ex1_10.wav
ex1_10st.wvc	- a corrupted version of ex1_10st.wav
ex1_10wp.wav	- a wavpack'd version of ex1_10.wav

ex2_01.wav	- a 1 Second artificially create waveform, in the PCM 01
		  byte form.  This file is the same as ex2_10.wav.
ex2_01st.wav	- a shorten'd version of ex2_01.wav
ex2_01wp.wav	- a wavpack'd version of ex2_01.wav
ex2_10.wav	- a 1 Second artificially create waveform, in the PCM 10
		  byte form.  This file is the same as ex2_10.wav.
ex2_10st.wav	- a shorten'd version of ex2_10.wav
ex2_10wp.wav	- a wavpack'd version of ex2_10.wav

ex4.wav		- a ulaw converted version of ex1_10.wav
ex4_01.wav	- a PCM 01 version of ex4.wav.  Note, this is NOT
		  expected to be the same as ex1_10.wav.
ex4_10.wav	- a PCM 10 version of ex4.wav.  Note, this is NOT
		  expected to be the same as ex1_10.wav.
ex4_sh.wav	- a shorten'd version of ex4.wav
ex4_wp.wav	- a wavpack'd version of ex4.wav

ex5.wav		- a 2 channel, 2 second,  ULAW file
ex5_sh.wav	- a shorten'd version of ex5.wav
ex5_wp.wav	- a wavpack'd version of ex5.wav
ex5_p.wav	- a 2 channel, 2 second,  PCM version of ex5.wav
		  (made by other utilities)
ex5_p01.wav	- a 01 byte order version of ex5_p.wav
ex5_p_sh.wav	- a shorten'd version of ex5_p.wav
ex5_p_wp.wav	- a wavpack'd version of ex5_p.wav
ex5_c1.wav	- the first channel of ex5.wav.
ex5_c1_p.wav	- a PCM version of ex5_c1.wav (made by other
		  utilities)
ex5_c2.wav	- the second channel of ex5.wav.
ex5_c2_p.wav	- a PCM version of ex5_c2.wav (made by other
		  utilities)
ex5_12.wav	- Channels 1 and 2 added version of ex5.wav
ex5_12_p.wav	- Channels 1 and 2 added version of ex5_p.wav

ex6.wav		- a 4 channel pcm data file, 8000 Khz, 1 sec.
		  channel 1: 12.5 Hz
		  channel 2: 25   Hz
		  channel 3: 50   Hz
		  channel 4: 100  Hz
		
ex7.wav		- a pculaw converted version of ex1_10.wav
ex7_01.wav	- a PCM 01 version of ex7.wav.  Note, this is NOT
		  expected to be the same as ex1_10.wav.
ex7_10.wav	- a PCM 10 version of ex7.wav.  Note, this is NOT
		  expected to be the same as ex1_10.wav.
ex7_sh.wav	- a shorten'd version of ex7.wav
ex7_wp.wav	- a wavpack'd version of ex7.wav
